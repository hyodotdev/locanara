# Locanara Documentation

![Locanara Logo](public/logo.png)

Locanara is a unified SDK for on-device AI capabilities across iOS (Apple Intelligence) and Android (Gemini Nano).

## Overview

Locanara provides a consistent, GraphQL-based API that abstracts platform-specific implementations. All AI processing happens on-device, ensuring privacy and offline functionality.

## Website

Visit [locanara.dev](https://locanara.dev) for full documentation.

## Features

- **Summarize** - Text summarization
- **Classify** - Text classification
- **Extract** - Entity extraction
- **Chat** - Conversational AI
- **Translate** - Language translation
- **Rewrite** - Text rewriting
- **Proofread** - Grammar correction
- **Describe Image** - Image descriptions

## Platform Support

- iOS 26+ with Apple Intelligence
- Android 14+ (API 34+) with Gemini Nano
