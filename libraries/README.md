# Libraries

Third-party framework integrations for Locanara SDK.

| Library | Status | Description |
|---------|--------|-------------|
| [expo-ondevice-ai](./expo-ondevice-ai) | In Progress | Expo module for on-device AI |
| [react-native-ondevice-ai](./react-native-ondevice-ai) | Planned | React Native module for on-device AI |

## expo-ondevice-ai

Expo module wrapping Locanara SDK for React Native/Expo apps.

```bash
cd libraries/expo-ondevice-ai
bun install
bun run build
bun run test
```

## react-native-ondevice-ai

React Native module (planned) - will provide direct React Native bindings without Expo dependency.
